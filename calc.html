ℹ️ Help & Guide

📌 How to use (ఉపయోగ విధానం)

  1. Enter Customer Name and Rate per KG in Entry Form
  2. Click Generate Table to create data entry rows
  3. Enter Weight (KG) and Bird Count for each row
  4. Add more rows with the ➕ Add Row button
  5. Use PDF Options tab to customize the report layout
  6. Click Generate PDF for a professional invoice
  7. Click Save Record to store in database history
💡 PDF Tips:
- The PDF uses the pre-loaded Ramesh Poultry Farms logo
- Upload a custom logo from the Entry tab to override
- Go to PDF Options tab to toggle columns, change colors, font size
- Supports 500+ rows with auto page breaks
